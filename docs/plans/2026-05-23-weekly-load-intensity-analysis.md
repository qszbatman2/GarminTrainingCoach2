# 本周训练量与训练强度 AI 分析规则

## 输入数据

### 一、活动级数据

1. 活动日期
2. 活动类型
3. 单次训练时长
4. 单次训练距离
5. 单次训练负荷 `trainingLoad`
6. 单次有氧训练效果 `aerobicTrainingEffect`
7. 单次无氧训练效果 `anaerobicTrainingEffect`
8. 单次平均心率、最高心率
9. 单次建议恢复时长 `recoveryHours`

### 二、日级汇总数据

1. 每日总强度分钟 `intensityMinutes`
2. 每日高强度分钟 `vigorousIntensityMinutes`
3. 7 天急性负荷 `ATL / acuteTrainingLoad`
4. 42 天慢性负荷 `CTL / chronicTrainingLoad`
5. 负荷比值 `ATL / CTL`
6. 睡眠评分、总睡眠、深睡、REM、睡眠中断
7. 静息心率 `RHR`
8. 心率变异性 `HRV`
9. 压力评分
10. 训练准备度 `trainingReadiness`
11. Body Battery 高点 / 低点
12. 夜间血氧
13. 最大摄氧量 `VO2Max`
14. 乳酸阈值心率
15. 体重

### 三、本版不纳入核心计算的字段

1. 身体水分
2. 皮肤温度 / 生病标签
3. 人工训练计划目标值

说明：

- 本文档只使用当前项目里已经能稳定提取或可降级提取的字段。
- 若 `trainingLoad`、`aerobicTrainingEffect`、`anaerobicTrainingEffect` 缺失，则强度分析自动降级为 `强度分钟 + 高强度分钟 + ATL/CTL` 方案。
- 若距离字段缺失或当前运动类型并非典型距离型项目，则训练量更侧重时长和训练次数。

## 详细运算规则

### 一、时间窗口定义

1. 参考日期：以当前最新同步日期为准
2. 本周窗口：本周一 00:00 至参考日期
3. 本月窗口：本月 1 日 00:00 至参考日期
4. 本周已过天数：

   本周已过天数 = 参考日期所在周从周一到今天的自然天数

5. 本月已过天数：

   本月已过天数 = 本月 1 日到参考日期的自然天数

### 二、月均值与周均值定义

1. 本月训练量日均：

   本月训练量日均 = 本月累计训练量 / 本月已过天数

2. 本月训练强度日均：

   本月训练强度日均 = 本月累计训练强度 / 本月已过天数

3. 本月训练量折算周均：

   本月训练量周均 = 本月训练量日均 × 7

4. 本月训练强度折算周均：

   本月训练强度周均 = 本月训练强度日均 × 7

说明：

- 这里的“本月周均值”采用“日均 × 7”的折算方式，而不是自然周切片均值。
- 这样可以避免月初、月末和不完整周造成统计偏差，更适合和“本周一到今天”的数据对比。

### 三、本周训练量计算逻辑

#### 1. 本周训练量原始指标

1. 本周总训练时长
2. 本周总训练距离
3. 本周训练次数

#### 2. 本月训练量基准

1. 本月总训练时长
2. 本月总训练距离
3. 本月训练次数
4. 本月训练量日均
5. 本月训练量周均

#### 3. 本周训练量进度比

1. 时长进度比：

   本周总训练时长 / (本月时长日均 × 本周已过天数)

2. 距离进度比：

   本周总训练距离 / (本月距离日均 × 本周已过天数)

3. 次数进度比：

   本周训练次数 / (本月训练次数日均 × 本周已过天数)

#### 4. 本周训练量预测全周值

1. 本周预测时长：

   本周总训练时长 / 本周已过天数 × 7

2. 本周预测距离：

   本周总训练距离 / 本周已过天数 × 7

3. 本周预测次数：

   本周训练次数 / 本周已过天数 × 7

#### 5. 本周训练量周均环比

1. 时长周均环比：

   本周预测时长 / 本月时长周均

2. 距离周均环比：

   本周预测距离 / 本月距离周均

3. 次数周均环比：

   本周预测次数 / 本月次数周均

#### 6. 训练量综合得分

当距离可用且当前运动以跑步、骑行、徒步等距离型项目为主时：

- 训练量得分 = 时长环比 × 0.45 + 距离环比 × 0.35 + 次数环比 × 0.20

当距离不可用或当前训练以力量、室内课程等非距离型项目为主时：

- 训练量得分 = 时长环比 × 0.70 + 次数环比 × 0.30

### 四、本周训练强度计算逻辑

#### 1. 本周训练强度原始指标

1. 本周总训练负荷 `trainingLoad` 累计
2. 本周平均单次训练负荷
3. 本周平均有氧训练效果
4. 本周平均无氧训练效果
5. 本周总强度分钟
6. 本周总高强度分钟

#### 2. 本月训练强度基准

1. 本月总训练负荷
2. 本月平均单次训练负荷
3. 本月平均有氧训练效果
4. 本月平均无氧训练效果
5. 本月总强度分钟
6. 本月总高强度分钟
7. 本月训练强度日均
8. 本月训练强度周均

#### 3. 本周训练强度进度比

1. 训练负荷进度比：

   本周总训练负荷 / (本月训练负荷日均 × 本周已过天数)

2. 有氧训练效果进度比：

   本周平均有氧训练效果 / 本月平均有氧训练效果

3. 无氧训练效果进度比：

   本周平均无氧训练效果 / 本月平均无氧训练效果

4. 高强度分钟进度比：

   本周总高强度分钟 / (本月高强度分钟日均 × 本周已过天数)

#### 4. 本周训练强度预测全周值

1. 本周预测训练负荷：

   本周总训练负荷 / 本周已过天数 × 7

2. 本周预测高强度分钟：

   本周总高强度分钟 / 本周已过天数 × 7

#### 5. 本周训练强度周均环比

1. 训练负荷周均环比：

   本周预测训练负荷 / 本月训练负荷周均

2. 高强度分钟周均环比：

   本周预测高强度分钟 / 本月高强度分钟周均

#### 6. 训练强度综合得分

当 `trainingLoad + trainingEffect` 可用时：

- 训练强度得分 = 训练负荷环比 × 0.45 + 有氧训练效果环比 × 0.20 + 无氧训练效果环比 × 0.15 + 高强度分钟环比 × 0.20

当活动级训练效果字段不足时：

- 训练强度得分 = 训练负荷环比 × 0.55 + 总强度分钟环比 × 0.20 + 高强度分钟环比 × 0.25

当 `trainingLoad` 也缺失时：

- 训练强度得分 = 总强度分钟环比 × 0.45 + 高强度分钟环比 × 0.35 + ATL/CTL 负荷比校正 × 0.20

### 五、训练量 / 强度分档规则

#### 1. 单项环比判定

- < 0.75：明显不足
- 0.75 ~ 0.90：偏低
- 0.90 ~ 1.15：合理
- 1.15 ~ 1.30：偏高
- > 1.30：明显偏高

#### 2. 训练量结论

- 训练量综合得分 < 0.75：训练量不足
- 0.75 ~ 0.90：训练量偏低
- 0.90 ~ 1.15：训练量合理
- 1.15 ~ 1.30：训练量偏高
- > 1.30：训练量过高

#### 3. 训练强度结论

- 训练强度综合得分 < 0.80：训练强度不足
- 0.80 ~ 0.95：训练强度偏低
- 0.95 ~ 1.15：训练强度合理
- 1.15 ~ 1.30：训练强度偏高
- > 1.30：训练强度过高

### 六、过度训练与训练不足判定规则

#### 1. 过度训练高风险

满足以下任意一条，判定为过度训练高风险：

1. `ATL / CTL > 1.5`
2. 训练量结论为“过高”且训练强度结论为“过高”
3. 训练强度综合得分 > 1.30，且最近恢复时长仍偏长
4. 本周高强度分钟环比 > 1.30，且 HRV、睡眠、RHR 中至少 2 项恢复信号变差

#### 2. 训练偏多

满足以下条件之一，判定为训练偏多：

1. 训练量为“偏高”，训练强度为“合理”
2. 训练量为“合理”，训练强度为“偏高”
3. `ATL / CTL` 落在 `1.20 ~ 1.50`，且恢复能力未明显下降

#### 3. 训练合理

同时满足以下条件，判定为训练合理：

1. 训练量为“合理”或“轻微偏高 / 轻微偏低”
2. 训练强度为“合理”或“轻微偏高 / 轻微偏低”
3. `ATL / CTL` 落在 `0.80 ~ 1.20`
4. 睡眠、HRV、RHR、压力等恢复信号没有明显恶化

#### 4. 训练不足

满足以下任意一条，判定为训练不足：

1. 训练量综合得分 < 0.75 且训练强度综合得分 < 0.80
2. 训练量明显不足，且 `ATL / CTL < 0.5`
3. 本周训练次数、总时长、训练负荷均显著低于本月节奏

### 七、恢复信号校正规则

本需求核心是“训练量 / 强度是否合理”，但最终判断必须结合恢复状态做校正。

#### 1. 恢复走弱信号

满足以下任意一项，可视为恢复走弱：

1. 睡眠评分 < 60
2. 睡眠中断 > 5
3. HRV 低于个人近期基准 15%
4. RHR 高于个人近期基准 5bpm
5. 压力评分处于高分段
6. 建议恢复时长 >= 24 小时

#### 2. 校正规则

1. 若训练量 / 强度显示“合理”，但恢复走弱明显，则最终结论下调为“当前负荷偏重”
2. 若训练量 / 强度显示“偏高”，且恢复走弱，则直接提升为“过度训练风险”
3. 若训练量 / 强度显示“偏低”，但恢复信号很差，则不判为“偷懒”，而判为“当前更需要恢复”

## 输出规则

1. 本周训练量结论：四字内简短结论
2. 本周训练强度结论：四字内简短结论
3. 总体判断：`训练不足 / 训练合理 / 训练偏多 / 过度风险`
4. 今日建议：一句短句
5. 原因分析：300 字以内，必须嵌入以下数据

   - 本周累计训练时长 / 距离 / 次数
   - 本月日均节奏或折算周均
   - 本周训练负荷 / 训练效果 / 高强度分钟
   - `ATL / CTL` 负荷比
   - 至少 1~2 个恢复信号

## AI 输出示例 1

本周训练量结论：合理

本周训练强度结论：合理

总体判断：训练合理

今日建议：维持当前节奏，按计划推进

原因分析：本周一到今天累计训练 4 次，总时长 238 分钟、总距离 31.6km，按当前进度折算全周后与本月周均基本一致。训练负荷累计 512，平均有氧训练效果 3.1，高强度分钟 46 分钟，强度环比落在合理区间。当前 ATL/CTL 为 1.06，没有明显超负荷迹象。睡眠评分与 HRV 未见明显恶化，说明训练量和训练强度整体匹配。

本周训练强度结论：过高

总体判断：过度风险

今日建议：立即下调强度，优先恢复

原因分析：本周一到今天仅 4 天已累计训练 5 次、总时长 356 分钟，折算全周后明显高于本月周均。训练负荷累计 798，平均有氧训练效果 3.9，无氧训练效果 2.8，高强度分钟已达 82 分钟，强度环比显著偏高。当前 ATL/CTL 为 1.54，已进入超负荷区间；同时睡眠评分和 HRV 走弱，说明当前训练量和训练强度已超过合理范围。

## AI 输出示例 3

本周训练强度结论：偏低

总体判断：训练不足

今日建议：恢复正常训练频率，不要继续拖

原因分析：本周一到今天累计仅训练 1 次，总时长 42 分钟，训练量进度明显低于本月日均节奏，折算全周后也低于本月周均。训练负荷仅 86，高强度分钟 6 分钟，强度明显不足。当前 ATL/CTL 为 0.46，说明近期负荷偏低。若睡眠、HRV 和静息心率没有明显异常，则当前更像训练执行不足，而不是恢复性休息。
