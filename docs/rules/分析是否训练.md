## 输入数据

静息心率、心率变异性、呼吸频率、皮肤压力值；总睡眠、深睡、REM 睡眠、睡眠中断、睡眠评分；每日步数、活动热量、久坐时长、压力评分；单次训练负荷、7 天急性负荷 ATL、42 天慢性负荷 CTL、负荷比值 ATL/CTL、训练效果、估算恢复时长；最大摄氧量、乳酸阈值心率、体重、身体水分

注：当前线上同步拿不到稳定的夜间血氧、训练准备度、Garmin 原生建议恢复时长，因此这些字段不再直接作为原始输入参与“是否训练”的规则判断。
其中恢复时长改为根据最近一次训练强度自行估算。

## 详细运算规则

### 一、个人基线计算逻辑

1. 筛选近 28 天无生病、无熬夜、无超大强度训练的有效日常数据
2. 计算各项指标**基线平均值**、基线标准差
3. 基准判定范围：基线值 ±1.5 倍标准差，超出即为指标异动

### 二、单指标异常等级判断规则

1. 静息心率 RHR

- 超出基线 + 5bpm：轻度疲劳异常
- 超出基线 + 10bpm：重度疲劳异常

2. 心率变异性 HRV

- 低于基线 15%：轻度疲劳异常
- 低于基线 30%：重度疲劳异常

3. 睡眠评分

- 60 分以下：睡眠恢复不足异常
- 50 分以下：严重睡眠异常

4. 睡眠中断次数

- 单日超 5 次：睡眠质量异常

5. 日间压力评分

- 高分段判定身心疲劳叠加异常

### 三、综合疲劳分计算（总分 0-100，固定权重）

1. 单项指标按偏离基线程度，换算为 0-100 分项得分
2. 加权计算公式

   综合疲劳分 = HRV 得分 ×0.35 + 睡眠得分 ×0.25 + 静息心率得分 ×0.15 + 负荷比得分 ×0.15 + 压力得分 ×0.10

3. 分数对应身体状态

- 80~100 分：恢复优秀
- 60~79 分：恢复良好
- 40~59 分：中度疲劳
- 20~39 分：重度疲劳
- 0~19 分：极度疲劳

### 四、训练负荷比值判定规则

负荷比 = 7 天急性负荷 ATL ÷ 42 天慢性负荷 CTL

若 Garmin 原始 ATL/CTL 不可用，则回退为近 7 天训练时长与近 42 天周均训练时长的代理比值，继续给出保守判断，但要在缺失数据中明确提示是代理值。

- 0.8~1.2：负荷均衡，训练状态健康
- >1.5：超负荷，过度训练风险极高
- <0.5：负荷偏低，体能提升不足

### 五、身体恢复能力判断规则

追踪高强度训练后 RHR、HRV 回归基线用时

- 24~48 小时恢复：恢复能力正常
- >72 小时未回归：身体恢复能力偏弱

### 六、估算恢复时长规则

当 Garmin 原生 `recoveryHours` 不可用时，使用最近一次训练活动的强度字段做分段估算恢复时长。

估算输入：

- 单次训练负荷 `trainingLoad`
- 单次有氧训练效果 `aerobicTrainingEffect`
- 单次无氧训练效果 `anaerobicTrainingEffect`
- 单次中等强度分钟 `moderateIntensityMinutes`
- 单次高强度分钟 `vigorousIntensityMinutes`
- 单次训练时长 `durationMin`
- 单次距离 `distanceKm`

估算规则：

- 很轻松短课：`duration <= 35` 且 `trainingLoad < 80` 且 `vigorous < 20` 且 `anaerobic < 1`，恢复时长 = `2h`
- 长距离耐力课：`duration >= 150` 或 `distance >= 70km`，恢复时长 = `36h`
- 短时但有刺激的质量课：`duration <= 90` 且满足 `trainingLoad >= 80` / `vigorous >= 20` / `anaerobic >= 2` / `aerobic >= 3` 之一，恢复时长 = `12h`
- 中长课且刺激明显：`duration > 90` 且满足 `trainingLoad >= 150` / `vigorous >= 40` / `anaerobic >= 2` / `aerobic >= 3.5` 之一，恢复时长 = `24h`
- 其他情况默认恢复时长 = `6h`

规则约束：

- 当前个人化恢复档位为：`2 / 6 / 12 / 24 / 36`

解释口径：

- 短课恢复明显更快，不再因为训练负荷稍高就被拉到 18~24 小时
- 长距离耐力课单独处理，避免因为缺少强度分钟而被低估
- 该值只作为规则引擎恢复校正信号，不伪装成 Garmin 原始建议值

### 七、最终训练决策判定逻辑

1. 出现任意一项重度异常指标 或 综合疲劳分 < 40 分：判定不宜训练
2. 综合疲劳分 40~59 分：判定降强度训练
3. 综合疲劳分 60~79 分：判定正常强度训练
4. 综合疲劳分 80 分及以上：判定可适度加量训练
5. 负荷比 > 1.5，无论分数高低，强制降低当日训练强度

## 输出字段

1. 是否应该训练：二字简短结论
2. 今日训练建议：短句精简指引
3. 原因分析：300 字以内，**嵌入实测数据、与基线偏差值、疲劳分值、负荷比值**，结合指标状态综合说明判定依据

## 带数据说明输出示例 1

是否应该训练：可训练

今日训练建议：按原定强度正常训练

原因分析：当日静息心率 48bpm，相较个人基线仅高出 2bpm，心率变异性为基线数值 93%，两项核心神经指标无明显偏差。昨日睡眠评分 78 分，睡眠中断 2 次，睡眠恢复质量达标。本次测算综合疲劳得分 75 分，处于恢复良好区间。7 天与 42 天训练负荷比值 1.08，落在均衡合理范围，无负荷堆积情况。日间压力数值偏低，过往训练后指标均可 48 小时内回归基线，身体恢复能力稳定，整体状态满足常规训练条件。

## 带数据说明输出示例 2

是否应该训练：慎训

今日训练建议：下调训练强度，减少运动量

原因分析：当日静息心率较基线上升 6bpm，心率变异性下降 18%，达到轻度疲劳标准。睡眠评分仅 56 分，夜间中断 6 次，睡眠恢复不足。综合疲劳得分 52 分，判定为中度疲劳。训练负荷比值 1.32，接近超负荷临界值，短期训练压力累积明显。多项生理指标偏离正常区间，身体未完成充分休整，贸然高强度训练易引发身体不适，因此需降低当日运动强度。
